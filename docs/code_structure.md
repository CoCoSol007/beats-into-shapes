# Code structure 
Below is the structure of our code to help you move around the project.


```
res/
├── assets/ # Images of the game used throughout the project
├── scripts/
│   ├── score/ # Score management
│   ├── scene_transition/ # Transition between scenes
│   ├── speed_up/ # The speed up animation
│   ├── beats_manager/ # The logic around the beat of the music
│   ├── pause/ # Code related to the pause menu
│   ├── tuto/ # The tuto logic
|   └── contants.gd
├── scenes/
│   ├── menu/
│   ├── credit_menu/
│   ├── levels/
│   │   ├── selection/ # Level 1
│   │   ├── heating/ # Level 2
│   │   ├── forging/ # Level 3
│   │   ├── cooling/ # Level 4
├── docs/ # All project documentation
├── musics/ # All musics of the game
├── README.md
├── LICENSE
└── project.godot
```
